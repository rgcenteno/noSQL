| Comando                                   | Explicación breve                                                   | Ejemplo de salida                                        |
| ----------------------------------------- | ------------------------------------------------------------------- | -------------------------------------------------------- |
| `HSET clave campo valor`                  | Inserta o actualiza un campo dentro de un hash.                     | `HSET myhash field1 "foo"`                                            |
| `HGET clave campo`                        | Obtiene el valor de un campo específico del hash.                   | `HGET myhash field2`                                                  |
| `HMSET clave campo1 valor1 campo2 valor2` | Inserta múltiples campos en un hash (obsoleto, usar HSET múltiple). | `HMSET myhash field1 "Hello" field2 "World"`                                                     |
| `HMGET clave campo1 campo2`               | Obtiene varios campos del hash.                                     | `HMGET myhash field1 field2 nofield`                                    |
| `HGETALL clave`                           | Devuelve todos los campos y valores del hash.                       | `HGETALL myhash`        |
| `HDEL clave campo`                        | Elimina uno o más campos del hash.                                  | `HDEL myhash field1`                                            |
| `HEXISTS clave campo`                     | Comprueba si un campo existe en el hash.                            | `HEXISTS myhash`                                            |
| `HKEYS clave`                             | Devuelve todos los campos del hash.                                 | `HKEYS myhash`                               |
| `HVALS clave`                             | Devuelve todos los valores del hash.                                | `HVALS myhash`                                    |
| `HLEN clave`                              | Devuelve el número de campos del hash.                              | `HLEN myhash`                                            |
| `HINCRBY clave campo incremento`          | Incrementa el valor numérico de un campo.                           | `HINCRBY myhash field 1`                                           |
| `HSCAN clave cursor`                      | Itera los campos del hash sin bloquear el servidor.                 | `HSCAN key cursor [MATCH pattern] [COUNT count] [NOVALUES]` |
