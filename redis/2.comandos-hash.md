# Comandos Redis - Hash

## `HSET clave campo valor`

Inserta o actualiza un campo dentro de un hash.

```redis
127.0.0.1:6379> HSET myhash field1 "foo"
(integer) 1
```

## `HGET clave campo`

Obtiene el valor de un campo específico del hash.

```redis
127.0.0.1:6379> HGET myhash field1
"foo"
```

## `HMSET clave campo1 valor1 campo2 valor2`

Inserta múltiples campos en un hash. Está obsoleto, se recomienda usar `HSET` con múltiples pares.

```redis
127.0.0.1:6379> HMSET myhash field1 "Hello" field2 "World"
OK
```

## `HMGET clave campo1 campo2`

Obtiene varios campos del hash.

```redis
127.0.0.1:6379> HMGET myhash field1 field2
1) "Hello"
2) "World"
```

## `HGETALL clave`

Devuelve todos los campos y valores del hash.

```redis
127.0.0.1:6379> HGETALL myhash
1) "field1"
2) "Hello"
3) "field2"
4) "World"
```

## `HDEL clave campo`

Elimina uno o más campos del hash.

```redis
127.0.0.1:6379> HDEL myhash field1
(integer) 1
```

## `HEXISTS clave campo`

Comprueba si un campo existe en el hash.

```redis
127.0.0.1:6379> HEXISTS myhash field1
(integer) 1
```

## `HKEYS clave`

Devuelve todos los campos del hash.

```redis
127.0.0.1:6379> HKEYS myhash
1) "field1"
2) "field2"
```

## `HVALS clave`

Devuelve todos los valores del hash.

```redis
127.0.0.1:6379> HVALS myhash
1) "Hello"
2) "World"
```

## `HLEN clave`

Devuelve el número de campos del hash.

```redis
127.0.0.1:6379> HLEN myhash
(integer) 2
```

## `HINCRBY clave campo incremento`

Incrementa el valor numérico de un campo.

```redis
127.0.0.1:6379> HINCRBY myhash contador 1
(integer) 1
```

## `HSCAN clave cursor MATH pattern`

Itera los campos del hash sin bloquear el servidor.

```redis
127.0.0.1:6379> HSCAN myhash 0 MATCH field* COUNT 10
1) "0"
2) 1) "field1"
   2) "Hello"
```
